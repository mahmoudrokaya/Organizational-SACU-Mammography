# Results Directory

This directory contains all outputs generated by the Organizational SACU Mammography framework.

The purpose of this directory is to provide a reproducible and organized structure for:

* Model evaluation
* Manuscript generation
* Reviewer-response preparation
* Figure creation
* Statistical reporting

## Directory Structure

```text
results/
├── tables/
├── figures/
└── reports/
```

## Workflow

```text
Raw Data
    ↓
Preprocessing
    ↓
Representation Construction
    ↓
Feature Extraction
    ↓
Baseline Modeling
    ↓
SACU Training
    ↓
Evaluation
    ↓
Results/
```

## Expected Outputs

### Tables

Structured CSV/XLSX outputs used in manuscripts.

### Figures

Publication-quality visualizations.

### Reports

Comprehensive experiment summaries and reproducibility reports.

## Reproducibility

All files in this directory should be generated automatically through repository scripts.

Manual editing is discouraged.

## Manuscript Mapping

The contents of this directory support:

* Dataset characterization tables
* Baseline comparisons
* SACU pathway analyses
* Threshold analyses
* Ablation studies
* Bootstrap confidence intervals
* Organizational interpretability analyses
* Coordination ablation experiments

## Recommended Archiving

Before publication, archive:

* Final tables
* Final figures
* Experiment reports

alongside the GitHub release and Zenodo deposition.
