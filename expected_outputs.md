# Expected Outputs

## Organizational-SACU-Mammography

This document describes the principal outputs generated by the Organizational-SACU-Mammography pipeline. The repository is organized so that each stage produces clearly defined artifacts that can be inspected independently and reused in downstream analyses.

All generated outputs are stored under the following directory structure:

```text
results/
├── tables/
├── figures/
└── reports/
```

The exact number and names of files may vary depending on the dataset and experimental configuration.
