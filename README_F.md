## Quality Requirements

All publication figures should:

* Be vector format when possible.
* Use readable font sizes.
* Maintain journal-quality resolution.
* Clearly label axes and legends.

## Preferred Formats

* PNG
* PDF
* SVG

SVG or PDF is recommended for publication submission.
